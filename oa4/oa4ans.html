Our solution to Outside Assignment 4 (in FIB.ANS): 

separate (Fibtest)
function Fib(N : in Positive) return Positive is
   Answer : Positive := 1;
begin
   if N > 2 then
      Answer := Fib(N - 1) + Fib(N - 2);
   end if;
   return Answer;
end Fib;

Your solution is probably quite similar to ours.  Perhaps you used else and didn't initialize Answer in the declarative region.  Perhaps you had some other minor variation.  Our solution is probably no “better” than yours, if the test driver said, “Congratulations, you completed the assignment!”
